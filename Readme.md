# Fungelet, a very convenient befunge IDE
## Installation
```bash
git clone https://github.com/lomnom/Fungelet/
cd Fungelet
git clone https://github.com/lomnom/Terminal/
mv Terminal/Term*.py .
rm -rf Terminal
python3 Run.py
```
